## 项目说明
- 每个项目独立隔离，内置 `index.html` 入口文件
- 在线访问异常时，检查：
  1. 项目文件夹内是否存在 `index.html` 文件
  2. GitHub Pages 是否已开启并部署到正确分支
  3. 静态资源是否使用相对路径（避免以 `/` 开头的绝对路径）
